后端管理单页框架-说明

引用文件：lib/helangSingle.js

调用方式：

$.helangSingle({
    url:"pages/index.html", //url:'默认打开页的地址'
    title:"首页" //title:'默认打开页的标题'
});

其它说明：

  1. 页面元素ID名称不能更改，分别为：helangNav(导航),helangTab(选项卡),helangFrame(框架集容器),helangHead(头部容器)
  2. 导航绑定打开地址，给元素添加类h_nav并给这个元素添加自定义属性data-url写上链接地址既可(参照本示例的写法)。
